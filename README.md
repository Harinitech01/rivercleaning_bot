# River Cleaning Bot

## Project Overview
River Cleaning Bot is an automated system designed to collect floating waste from rivers, lakes, and water bodies. The main objective of this project is to reduce water pollution by using a robotic system that can detect, collect, and store waste materials such as plastic bottles, wrappers, and other floating debris.

This project focuses on environmental sustainability by combining embedded systems, automation, and smart waste management.

---

## Problem Statement
Water pollution caused by floating waste is a major environmental issue. Manual cleaning of rivers is time-consuming, expensive, and sometimes unsafe.

This project aims to develop an automated robotic solution that can:
- Detect floating waste
- Collect waste efficiently
- Reduce human effort
- Support cleaner water ecosystems

---

## Objectives
- Build an automated bot for cleaning floating waste in rivers
- Reduce manual labor involved in water cleaning
- Improve waste collection efficiency
- Develop a low-cost and eco-friendly solution

## Working Principle
1. The bot moves through the water surface.
2. Sensors detect obstacles or waste materials.
3. The conveyor mechanism collects floating waste.
4. Waste is transferred into a storage container.
5. The bot continues movement until stopped or storage is full.
